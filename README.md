# Ponte Legado

[![CI](https://github.com/GuilhermeCostaProenca/ponte-legado/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/GuilhermeCostaProenca/ponte-legado/actions/workflows/ci.yml)

Digital product repository focused on a real market pain with portfolio-grade execution.

## Proposal
**Modernizacao incremental de sistemas antigos** for Empresas medias e grandes com sistemas legados criticos.

## Problem
Sistemas antigos bloqueiam evolucao digital, mas migracao total e cara e arriscada.

## Target audience
Empresas medias e grandes com sistemas legados criticos

## Suggested stack
Backend .NET/Java, API gateway, PostgreSQL, frontend React

## Initial MVP
- Conector para bancos legados e ERPs antigos
- Camada de APIs modernas sobre estruturas existentes
- Painel web para operacoes criticas
- Monitoramento de latencia, falhas e uso por endpoint

## Base structure
- `README.md`: product vision and execution strategy
- `PRD.md`: functional scope and product rules
- `ARCHITECTURE.md`: initial technical design
- `BACKLOG.md`: prioritized execution backlog
- `PLAN.md`: phase-by-phase execution plan
- `.github/workflows/ci.yml`: validation pipeline

## How to start
1. Refine PRD with user and scope.
2. Confirm architecture and final stack.
3. Build core flow end-to-end.
4. Add CI, tests and operational docs.

## Main risks
Complexidade de integracao e heterogeneidade dos legados
